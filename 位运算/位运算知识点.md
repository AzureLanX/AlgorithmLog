## 一、六大基础操作

| 运算符 | 名称 | 规则 | 记忆口诀 |
|---|---|---|---|
| `&` | 与 | 同 1 才 1 | 都真才真 |
| `\|` | 或 | 有 1 就 1 | 有真就真 |
| `^` | 异或 | 不同才 1 | 不同为 1 |
| `~` | 取反 | 0 变 1,1 变 0 | 黑白颠倒 |
| `<<` | 左移 | 末尾补 0 | 等价乘 2ⁿ |
| `>>` | 右移 | 补符号位 | 等价除 2ⁿ(向下取整) |

```python
a = 5      # 0101
b = 3      # 0011

a & b  = 1 # 0001 - 都在的位置
a | b  = 7 # 0111 - 至少一个在
a ^ b  = 6 # 0110 - 不同的位置
~a     = -6 # 按位取反(Python 中结果是 -a-1)
a << 1 = 10 # 1010 - 乘 2
a >> 1 = 2  # 0010 - 除 2
```

## 二、必须背熟的核心性质

```python
# 异或
a ^ a = 0      # 自己和自己的异或是 0
a ^ 0 = a      # 异或 0 不变
交换律: a ^ b = b ^ a
结合律: (a ^ b) ^ c = a ^ (b ^ c)

# 与
a & a = a
a & 0 = 0
a & (~a) = 0   # 一个数和它的取反没有公共位

# 或
a | a = a
a | 0 = a
```

> **理解这些性质 = 解开一半位运算题。**

## 三、高频技巧(代码模板级别)

### 1. 判断奇偶

```python
if n & 1 == 0:   # 末位是 0 → 偶数
    pass
```

### 2. 判断 2 的幂(LC231)

```python
def is_power_of_two(n):
    return n > 0 and (n & (n - 1)) == 0
```

**原理**:2 的幂只有 1 个 1,减 1 后这一位变 0,后面全变 1,再与运算得 0。
例如 `8 = 1000`,`7 = 0111`,`8 & 7 = 0`。

### 3. 统计 1 的位数(LC191 / LC338)

```python
def hamming_weight(n):
    count = 0
    while n:
        n = n & (n - 1)   # 每次消掉最低位的 1
        count += 1
    return count
```

### 4. 提取最低位的 1(lowbit)— 树状数组核心

```python
lowbit = n & (-n)
# 6 = 0110,lowbit = 0010 = 2
# 12 = 1100,lowbit = 0100 = 4
```

### 5. 去掉最低位的 1

```python
n = n & (n - 1)
# 12 = 1100 → 1000 = 8
```

### 6. 获取某一位

```python
# 第 k 位(从 0 开始,0 是最低位)
bit = (n >> k) & 1
```

### 7. 设置某一位为 1 / 0 / 翻转

```python
n | (1 << k)    # 第 k 位设 1
n & ~(1 << k)   # 第 k 位设 0
n ^ (1 << k)    # 第 k 位翻转
```

### 8. 异或交换两数(不用临时变量)

```python
a ^= b
b ^= a
a ^= b
# 面试偶尔考,实际工作中几乎不用
```

## 五、经典套路总结

### 套路 1:出现奇数次 → 异或

> 一堆数中只有一个出现奇数次,其他偶数次 → 全异或

```python
def single_number(nums):
    result = 0
    for n in nums:
        result ^= n
    return result
```

### 套路 2:两个出现奇数次的数 → 分组异或(LC260)

```python
# 1. 全异或得到 a ^ b(不为 0)
# 2. 找 a ^ b 中最低位的 1(lowbit)
# 3. 按该位是 0/1 分成两组
# 4. 各自组内异或 → 得到 a 和 b
```

### 套路 3:不让你用加减乘除 → 位运算加法(LC371)

```python
def add(a, b):
    while b != 0:
        carry = (a & b) << 1   # 进位
        a = a ^ b              # 不进位相加
        b = carry
    return a
```

### 套路 4:子集枚举

```python
# 数组 [a, b, c] 的所有子集
n = 3
for mask in range(1 << n):     # 0 ~ 7
    for i in range(n):
        if mask & (1 << i):    # 第 i 位是 1,选 a[i]
            ...
```

## 六、必须记住的边界

```python
# Python 中负数用补码表示,~n 不是 -(n+1) 的简单取反
~5 = -6      # 5 = ...00000101,~5 = ...11111010 → -6

# 右移 >> 在 Python 中是算术右移(补符号位)
-1 >> 1 = -1 # 全是 1,右移还是全 1
# Java/C++ 中还有 >>> 逻辑右移,Python 没有

# Python 整数没有 32 位限制,处理 32 位相关题要手动 & 0xFFFFFFFF
n = n & 0xFFFFFFFF
```

## 七、面试常考易错点

1. **优先级**:`<<`、`>>` 优先级低于 `+`、`-`,要加括号:`(1 << k) - 1`
2. **符号位**:Python 没有 32 位限制,负数右移不会变 0
3. **原地修改**:位运算经常用来做 O(1) 空间修改
4. **不要死记**:每道题先想「为什么」再写代码
