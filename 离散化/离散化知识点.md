# 离散化知识点大全

> 把「稀疏大范围」数据压缩到「密集小范围」下标

---

## 一、什么是离散化?

**问题场景**:数组下标范围 `0 ~ 10^9`,但**实际只用到 10^5 个值**。直接开 10^9 大小的数组不可能。

**离散化**:把这些稀疏的「大」值,映射到「小」的下标(0, 1, 2, ...)。

```
原始值(稀疏):  [1, 1000000, 5, 999999999, 1, 5]
                  ↓ 排序去重 + 重新编号
离散化映射:    {1: 1, 5: 2, 1000000: 3, 999999999: 4}
```

**核心性质**:**保留相对大小关系**。
- 1 < 5 → 1 < 2
- 5 < 1000000 → 2 < 3
- 不会改变「谁大谁小」

---

## 二、离散化的标准三步

```
① 收集所有需要离散化的值(原始数据 + 查询涉及的值)
② 排序 + 去重
③ 用二分查找,把原值映射到下标
```

### Python 标准模板

```python
import bisect

def discretize(values):
    """把 values 列表离散化,返回排序去重后的数组"""
    return sorted(set(values))

def query(x, sorted_unique):
    """x 在 sorted_unique 中的下标(从 1 开始)"""
    return bisect.bisect_left(sorted_unique, x) + 1
```

### 用字典查(更快,O(1))

```python
def discretize(values):
    sorted_unique = sorted(set(values))
    return {v: i+1 for i, v in enumerate(sorted_unique)}

# 查询
def query(x, mp):
    return mp[x]   # O(1) 字典查询
```

## 三、为什么下标从 1 开始?

为了配合**前缀和**(前缀和数组 `s[0] = 0` 是哨兵)。

```python
a = [离散化后的值]   # 1-indexed
s = [0] * (n + 1)
for i in range(1, n+1):
    s[i] = s[i-1] + a[i]
# 区间 [l, r] 之和 = s[r] - s[l-1]
```

## 四、经典应用:ac802 离散化

### 题目

```
假定有一个无限长的数轴,数轴上每个坐标上的数都是 0。
现在我们首先进行 n 次操作,每次操作将某一位置 x 上的数加 c。
接下来,进行 m 次询问,每个询问包含两个整数 l 和 r,
你需要求出在区间 [l, r] 之间的所有数的和。
```

**关键**:x 的范围 `0 ~ 10^9`,但 n + 2m ≤ 10^5,**必须离散化**。

### 标准解法

```python
import bisect

def main():
    n, m = map(int, input().split())

    # 1. 收集所有需要离散化的坐标
    xs = []
    ops = []  # 存原始操作
    for _ in range(n):
        x, c = map(int, input().split())
        xs.append(x)
        ops.append((x, c))

    queries = []  # 存原始查询
    for _ in range(m):
        l, r = map(int, input().split())
        xs.append(l)
        xs.append(r)
        queries.append((l, r))

    # 2. 排序去重
    sorted_xs = sorted(set(xs))

    # 3. 把原值映射到下标
    # 用字典加速
    mp = {v: i+1 for i, v in enumerate(sorted_xs)}

    # 4. 在离散化数组上做操作
    N = len(sorted_xs) + 2   # 多开 1 位防越界
    a = [0] * (N + 1)
    for x, c in ops:
        a[mp[x]] += c

    # 5. 算前缀和
    s = [0] * (N + 1)
    for i in range(1, N + 1):
        s[i] = s[i-1] + a[i]

    # 6. 回答查询
    for l, r in queries:
        print(s[mp[r]] - s[mp[l] - 1])

main()
```

## 七、必须背熟的小技巧

### 1. 「要离散化哪些值」要全

```python
# ❌ 只离散化操作中的 x
xs = [x for x, c in ops]

# ✅ 还要加上查询中的 l 和 r
xs = [x for x, c in ops]
for l, r in queries:
    xs.append(l)
    xs.append(r)
```

### 2. 用 `bisect_left` 还是 `bisect_right`?

```python
# 离散化后,值是连续的
# bisect_left:找第一个 >= x 的位置 → 1-indexed 下标
# bisect_right:找第一个 > x 的位置

# 几乎所有场景用 bisect_left 就对了
```

### 3. 字典 vs 二分

```python
# 字典:预处理 O(n) + 查询 O(1)  ← 推荐
mp = {v: i+1 for i, v in enumerate(sorted_xs)}

# 二分:不需要预处理 + 查询 O(log n)
import bisect
idx = bisect.bisect_left(sorted_xs, x) + 1
```

**多次查询用字典,偶尔查询用二分**。

### 4. 「开大一点」防越界

```python
N = len(sorted_xs) + 2   # 差分场景必加
a = [0] * (N + 1)
```

### 5. 离散化不改变「相等」关系

```python
[1, 5, 1000000]  →  [1, 2, 3]   # 1<5 → 1<2 ✓
```

但**相等的值必须映射到同一个下标**,所以**必须去重**。

## 八、离散化的两种思维

### 思维 1:保序映射(最常用)

```
原值 → 下标,保持原值的大小关系
[1, 5, 10^9] → [1, 2, 3]
```

### 思维 2:值域压缩(高级)

```
把 [l, r] 区间内的所有整数看作一个「点」
l, r 都加到数组里,排序去重
```

---

## 九、常见陷阱

1. **漏加查询值**:只加了操作坐标,没加查询的 l、r → 查询结果错
2. **没去重**:相等的值映射到不同下标,会算错
3. **下标从 0 开始**:前缀和场景会少算第一个
4. **开小了**:差分需要 `n + 2`,普通前缀和 `n + 1` 就够
5. **bisect 用错**:用 `bisect_left` 还是 `bisect_right` 影响结果

## 十、离散化模板(通用版)

```python
import bisect
from typing import List

class Discretizer:
    def __init__(self, values: List):
        self.sorted_unique = sorted(set(values))
        self.mp = {v: i+1 for i, v in enumerate(self.sorted_unique)}

    def query(self, x):
        """返回 x 的离散化下标(1-indexed)"""
        return self.mp[x]

    def query_bisect(self, x):
        """二分版(应对 x 不在原 values 中的情况)"""
        return bisect.bisect_left(self.sorted_unique, x) + 1

    def size(self):
        return len(self.sorted_unique)
```

**使用**:
```python
ds = Discretizer([1, 5, 1000000])
print(ds.query(5))         # 2
print(ds.query_bisect(3))  # 2(3 应该在 1 和 5 之间)
```

## 十一、ac802 走读

**输入**:
```
3 3
1 2
3 6
7 5
1 3
4 7
8 8
```

**步骤**:
```
① 收集坐标:  [1, 3, 7, 1, 3, 4, 7, 8, 8]
② 排序去重:  [1, 3, 4, 7, 8]
③ 映射:      1→1, 3→2, 4→3, 7→4, 8→5
④ 操作:
   a[1] += 2  (x=1)
   a[2] += 6  (x=3)
   a[4] += 5  (x=7)
   a = [0, 2, 6, 0, 5, 0]
⑤ 前缀和:   s = [0, 2, 8, 8, 13, 13]
⑥ 查询:
   [1,3]:s[2] - s[0] = 8
   [4,7]:s[4] - s[2] = 13 - 8 = 5
   [8,8]:s[5] - s[4] = 13 - 13 = 0
```
