## 一、前缀和(Prefix Sum)

### 1. 什么是前缀和?

```
原数组:    nums   = [a, b, c, d, e]
前缀和数组: prefix = [0, a, a+b, a+b+c, a+b+c+d, a+b+c+d+e]
             ↑
           prefix[0] = 0(哨兵,让计算更统一)
```

**定义**:`prefix[i]` = 原数组前 i 个元素之和(下标 0 到 i-1)

```python
prefix[0] = 0
prefix[1] = nums[0]
prefix[2] = nums[0] + nums[1]
prefix[i] = prefix[i-1] + nums[i-1]
```

### 2. 前缀和的核心用途 — O(1) 求区间和

```
求 nums[l..r] 的和(闭区间,包含两端):

       l        r
       ↓        ↓
nums:  [a, b, c, d, e]
prefix:[0, a, a+b, a+b+c, a+b+c+d, a+b+c+d+e]
         ↑  ↑             ↑
        pre[l]          pre[r+1]

区间和 = prefix[r+1] - prefix[l]
```

```python
# 闭区间 [l, r] 的和
def range_sum(prefix, l, r):
    return prefix[r + 1] - prefix[l]
```

**对比**:
- 不用前缀和:每次 O(n) 遍历
- 用前缀和:预处理 O(n),**每次查询 O(1)**

### 3. 标准模板

```python
class NumArray:
    def __init__(self, nums):
        n = len(nums)
        self.prefix = [0] * (n + 1)     # 长度 +1,prefix[0] = 0
        for i in range(n):
            self.prefix[i + 1] = self.prefix[i] + nums[i]

    def sumRange(self, left, right):
        return self.prefix[right + 1] - self.prefix[left]
```

## 二、差分(Difference Array)— 前缀和的「逆运算」

### 1. 什么是差分?

```
原数组:    nums   = [a, b, c, d, e]
差分数组:  diff   = [a, b-a, c-b, d-c, e-d]
             ↑
           diff[0] = nums[0]
           diff[i] = nums[i] - nums[i-1]
```

**性质**:对差分数组求**前缀和**,就能还原原数组!

```python
# 由差分还原原数组
nums = [0] * n
for i in range(n):
    nums[i] = nums[i-1] + diff[i]   # i=0 时 nums[-1] 视为 0
```

### 2. 差分的核心用途 — O(1) 区间修改

**经典问题**:把数组的 `[l, r]` 区间内所有元素都加 v,要执行很多次这样的操作,最后问数组长什么样?

**暴力**:每次操作 O(n),k 次就是 O(nk)

**差分解法**:**O(1) 修改一次,前缀和还原 O(n)**,总共 O(n + k)

```python
# 初始化差分数组
diff = [0] * n
diff[0] = nums[0]
for i in range(1, n):
    diff[i] = nums[i] - nums[i-1]

# 区间 [l, r] 加 v
diff[l] += v
if r + 1 < n:
    diff[r + 1] -= v

# 还原原数组
for i in range(1, n):
    nums[i] = nums[i-1] + diff[i]
```

### 3. 为什么 `diff[l] += v, diff[r+1] -= v`?

```
原数组:    [a, b, c, d, e]
                  ↑       ↑
                  l       r
对 [l, r] 加 v,变成:
            [a, b, c+v, d+v, e+v]

差分数组变化:
diff[l-1] 不变(还是 a)
diff[l]   多了 v  ←  从 l 开始,所有后续前缀和都 +v
diff[r]   不变
diff[r+1] 少了 v  ←  从 r+1 开始,前缀和抵消回来
```

前缀和累加时,从 `l` 起到 `r` 的部分会持续 +v,到了 `r+1` 被减回去,**精准只影响 [l, r]**。

### 4. 标准模板

```python
# LC 1109 航班预订统计
class Solution:
    def corpFlightBookings(self, bookings, n):
        diff = [0] * n
        for l, r, v in bookings:       # l, r 是 1-indexed
            diff[l-1] += v
            if r < n:
                diff[r] -= v
        # 还原
        for i in range(1, n):
            diff[i] += diff[i-1]
        return diff
```

## 三、前缀和 + 哈希表:O(n) 解子数组问题

### LC560:和为 K 的子数组

**题目**:求和恰好为 k 的连续子数组个数

**朴素**:O(n²) 枚举所有 [l, r]

**优化**:前缀和 + 哈希表

**核心等式**:
```
nums[l..r] 之和 = k
⟺ prefix[r+1] - prefix[l] = k
⟺ prefix[l] = prefix[r+1] - k
```

**思路**:从左到右扫描,看 `prefix[i] - k` 在前面出现过几次,累加即可。

```python
from collections import defaultdict

class Solution:
    def subarraySum(self, nums, k):
        count = 0
        prefix = 0
        seen = defaultdict(int)
        seen[0] = 1            # 空前缀(选空子数组的情况)

        for num in nums:
            prefix += num
            count += seen[prefix - k]   # 找之前有多少 prefix[j] = prefix - k
            seen[prefix] += 1
        return count
```

**复杂度**:O(n) 时间,O(n) 空间

## 四、二维前缀和

### 1. 定义

```
pre[i][j] = matrix[0..i-1][0..j-1] 矩形区域内所有元素之和
```

`(i, j)` 位置代表「前 i 行、前 j 列」的矩形和。

### 2. 构造

```python
def build_pre(matrix):
    m, n = len(matrix), len(matrix[0])
    pre = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            # 整个大矩形 = 上 + 左 - 重叠左上 + 当前格
            pre[i][j] = pre[i-1][j] + pre[i][j-1] - pre[i-1][j-1] + matrix[i-1][j-1]
    return pre
```

### 3. 查询子矩阵和

```
求 (r1, c1) 到 (r2, c2) 矩形区域的和(都是闭区间):

    pre[r2+1][c2+1]                       右下角外的全和
  - pre[r1][c2+1]                         减掉 r1 行以上
  - pre[r2+1][c1]                         减掉 c1 列以左
  + pre[r1][c1]                           上面两个减了两次,加回来
```

```python
def query(pre, r1, c1, r2, c2):
    return (pre[r2+1][c2+1]
            - pre[r1][c2+1]
            - pre[r2+1][c1]
            + pre[r1][c1])
```

### 4. 直观图示

```
          c1     c2
    +-----+------+--+
    |  A  |  B   |  |  r1
    +-----+------+--+
    |  C  |  D   |  |  (要查的区域 D)
    +-----+------+--+ r2
    |     |      |  |
    +-----+------+--+

D = 整块 - A - C - B + (A+C)∩(A+B) = 容斥原理
  = pre[r2+1][c2+1] - pre[r1][c2+1] - pre[r2+1][c1] + pre[r1][c1]
```

## 五、二维差分

### 1. 构造

```python
# matrix 的差分
diff = [[0] * (n + 1) for _ in range(m + 1)]
for i in range(m):
    for j in range(n):
        diff[i+1][j+1] = (matrix[i][j]
                          - matrix[i-1][j] if i > 0 else 0
                          - matrix[i][j-1] if j > 0 else 0
                          + matrix[i-1][j-1] if i > 0 and j > 0 else 0)
```

### 2. 区间修改(子矩阵加 v)

```python
# 子矩阵 (x1, y1) 到 (x2, y2) 全部加 v
diff[x1][y1] += v
diff[x2+1][y1] -= v
diff[x1][y2+1] -= v
diff[x2+1][y2+1] += v
```

### 3. 还原

对 `diff` 做二维前缀和即可还原回 matrix。

## 七、必须背熟的小技巧

### 1. 哨兵 prefix[0] = 0

```python
prefix = [0] * (n + 1)   # 多开一位,prefix[0] = 0
```

**好处**:统一所有区间计算,不用特判 `l == 0` 的情况。

### 2. 闭区间 vs 半开区间

| 区间写法 | 公式 |
|---|---|
| 闭区间 `[l, r]`(都包含) | `prefix[r+1] - prefix[l]` |
| 半开 `[l, r)`(只含 l) | `prefix[r] - prefix[l]` |

### 3. 前缀和判断「中心下标」

```python
# LC724:左边元素之和 = 右边元素之和
for i in range(n):
    if prefix[i] == total - prefix[i] - nums[i]:
        return i
```

### 4. 差分适用场景

**特征**:
- 多次「区间加 v」操作
- 最后才查询一次

**不适用**:
- 边修改边查询 → 用线段树/树状数组

### 5. 前缀和的「倒着用」

有时题目给的是「**后缀**和」数组,需要反向操作:

```python
# 后缀和:suffix[i] = nums[i] + nums[i+1] + ... + nums[n-1]
suffix = [0] * (n + 1)
for i in range(n-1, -1, -1):
    suffix[i] = suffix[i+1] + nums[i]
```

## 八、易错点

1. **下标偏移**:闭区间 `[l, r]`,公式用 `prefix[r+1] - prefix[l]`
2. **边界判断**:差分修改时,`r+1 < n` 才需要 `diff[r+1] -= v`
3. **二维前缀和 4 项公式**:一定要加回左上角那块(被减了两次)
4. **哈希表初始化**:`seen[0] = 1`,处理「从下标 0 开始的子数组」
5. **看清题目区间**:`[l, r]` 还是 `[l, r)`,索引是 0-indexed 还是 1-indexed
