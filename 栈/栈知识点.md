## 一、栈是什么?

**栈(Stack)**:后进先出(LIFO)的容器,像一摞盘子。

```
        ┌───┐
        │ 5 │  ← 栈顶(刚放进去的,最先出去)
        ├───┤
        │ 4 │
        ├───┤
        │ 3 │
        ├───┤
        │ 2 │
        ├───┤
        │ 1 │  ← 栈底(最先放进去的,最后才能出去)
        └───┘
```

## 二、基本操作

| 操作 | 说明 | 时间 |
|---|---|---|
| `push` / `append` | 压栈 | O(1) |
| `pop` / `pop()` | 出栈 | O(1) |
| `top` / `peek` / `[-1]` | 看栈顶 | O(1) |
| `empty` / `not stack` | 判空 | O(1) |
| `size` / `len` | 大小 | O(1) |

## 三、Python 中的栈

```python
# 1. 用 list 当栈(常用)
stack = []
stack.append(x)         # 压栈
stack.pop()             # 出栈(不传参,弹最后一个)
stack[-1]               # 看栈顶
not stack               # 判空

# 2. collections.deque(更快,推荐)
from collections import deque
stack = deque()
stack.append(x)
stack.pop()
stack[-1]
not stack

# 区别:deque 的 pop() 是 O(1),list 严格说也是 O(1) 分摊
```

## 四、栈的 5 大经典应用

### 1. 括号匹配(LC20)

**核心**:左括号入栈,右括号出栈,看是否匹配。

```python
def isValid(s):
    stack = []
    pairs = {')': '(', ']': '[', '}': '{'}
    for c in s:
        if c in '([{':          # 左括号:入栈
            stack.append(c)
        else:                   # 右括号:看是否匹配
            if not stack or stack[-1] != pairs[c]:
                return False
            stack.pop()
    return not stack
```

**关键**:
- 最后栈必须**空**
- 右括号出现时栈不能空
- 类型必须匹配

### 2. 表达式求值(LC150 逆波兰)

**逆波兰(RPN)**:操作符在操作数**后面**。

```
中缀:    (1 + 2) × 3
逆波兰:  1 2 + 3 *
```

```python
def evalRPN(tokens):
    stack = []
    for t in tokens:
        if t in '+-*/':
            b, a = stack.pop(), stack.pop()    # 注意顺序!
            if t == '+': stack.append(a + b)
            elif t == '-': stack.append(a - b)
            elif t == '*': stack.append(a * b)
            else: stack.append(int(a / b))     # Python 除法要特殊处理
        else:
            stack.append(int(t))
    return stack[0]
```

**关键**:
- 操作符时,弹两个,先弹的是**第二个操作数 b**
- Python `/` 是真除,题目一般要**截断**(用 `int(a/b)` 而不是 `a//b`,负数时不同)

### 3. 字符串解码(LC394)

```
s = "3[a2[c]]" → "accaccacc"
```

```python
def decodeString(s):
    stack = []
    num = 0
    for c in s:
        if c.isdigit():
            num = num * 10 + int(c)
        elif c == '[':
            stack.append((num, ''))     # (重复次数, 当前串)
            num = 0
        elif c == ']':
            k, prev = stack.pop()
            stack[-1] = (stack[-1][0], stack[-1][1] + prev * k)
        else:                           # 字母
            stack[-1] = (stack[-1][0], stack[-1][1] + c)
    return stack[-1][1]
```

### 4. 模拟递归

任何递归都能改成栈,避免栈溢出。

```python
# 递归
def factorial(n):
    if n <= 1: return 1
    return n * factorial(n - 1)

# 栈模拟
def factorial(n):
    stack = [n]
    res = 1
    while stack:
        x = stack.pop()
        if x <= 1: continue
        res *= x
        stack.append(x - 1)
    return res
```

## 五、单调栈(核心中的核心)

### 什么是单调栈?

栈里的元素**保持单调**(递增或递减)。

```
数组:    [2, 1, 5, 6, 2, 3]
下一个更大元素:   [5, 5, 6, -1, 3, -1]

遍历到 2 (i=0):栈空,2 入栈 → [2]
遍历到 1 (i=1):1 < 2,1 入栈 → [2, 1]
遍历到 5 (i=2):5 > 1,弹出 1,记 result[1]=5
                5 > 2,弹出 2,记 result[0]=5
                5 入栈 → [5]
遍历到 6 (i=3):6 > 5,弹出 5,记 result[2]=6
                6 入栈 → [6]
遍历到 2 (i=4):2 < 6,2 入栈 → [6, 2]
遍历到 3 (i=5):3 > 2,弹出 2,记 result[4]=3
                3 入栈 → [6, 3]
栈非空,继续弹,记 result[3]=-1,result[5]=-1
```

### 单调栈模板(下一个更大元素)

```python
def nextGreater(nums):
    n = len(nums)
    res = [-1] * n
    stack = []                # 存「下标」

    for i in range(n):
        # 当栈顶元素 < 当前元素,栈顶的「下一个更大」就是当前
        while stack and nums[stack[-1]] < nums[i]:
            idx = stack.pop()
            res[idx] = nums[i]
        stack.append(i)

    return res
```

### 变形:下一个更小元素

把 `<` 改成 `>`。

### 变形:上一个更大元素

从**左往右**遍历(改成 `i` 倒着,或换个判断方向)。

### 变形:循环数组(LC503)

数组是循环的,把数组**翻倍**,或者用 `i % n` 取模遍历 2n 次。

```python
def nextGreaterElements(nums):
    n = len(nums)
    res = [-1] * n
    stack = []
    for i in range(2 * n):            # 遍历两遍
        while stack and nums[stack[-1] < nums[i % n]]:
            res[stack.pop()] = nums[i % n]
        if i < n:                     # 只在第一遍入栈
            stack.append(i)
    return res
```


### 4. 单调栈:while-pop,不是 if


### 5. 路径类问题:用栈处理 `..` 和 `.`

```python
for token in path.split('/'):
    if token == '..':
        if stack: stack.pop()
    elif token and token != '.':
        stack.append(token)
```

### 6. 最小栈:辅助栈同步

```python
class MinStack:
    def __init__(self):
        self.stack = []
        self.min_stack = [float('inf')]
    
    def push(self, x):
        self.stack.append(x)
        self.min_stack.append(min(x, self.min_stack[-1]))
    
    def pop(self):
        self.stack.pop()
        self.min_stack.pop()
    
    def top(self):
        return self.stack[-1]
    
    def getMin(self):
        return self.min_stack[-1]
```

## 八、常见陷阱

1. **pop 两次时顺序**:b 在前,a 在后
2. **空栈 pop**:`stack[-1]` 或 `stack.pop()` 前要 `if stack`
3. **Python `/` 和 `//`**:负数除法行为不同
4. **单调栈忘初始化 res**:默认填 -1 而不是 0
5. **最小栈忘同步**:辅助栈必须和主栈同步 push/pop
6. **递归转栈忘保存状态**:局部变量要存到栈里

## 九、栈 vs 队列

| | 栈 | 队列 |
|---|---|---|
| 顺序 | LIFO(后进先出) | FIFO(先进先出) |
| 类比 | 一摞盘子 | 排队 |
| 操作 | push/pop | push/pop(从另一头) |
| 应用 | 递归、DFS、括号 | BFS、滑动窗口 |
