```python
# 一个区间:用 [start, end] 表示(也写作 [l, r])
```

## 二、核心套路 1:合并区间(LC56)

### 1. 题目

```
输入:[[1,3],[2,6],[8,10],[15,18],[16,19]]
输出:[[1,6],[8,10],[15,19]]
```

### 2. 标准思路(三步走)

```
① 按左端点排序
② 遍历,维护「当前合并区间」cur
③ 判断下一个区间与 cur 的关系:
   - 不重叠:把 cur 加入结果,cur 更新为下一个区间
   - 重叠  :合并 → cur 的右端点 = max(cur.end, next.end)
```

### 3. 标准代码

```python
def merge(intervals):
    if not intervals:
        return []

    intervals.sort(key=lambda x: x[0])     # ① 按左端点排序
    res = []
    cur = intervals[0]

    for nxt in intervals[1:]:
        if nxt[0] > cur[1]:                # 不重叠
            res.append(cur)
            cur = nxt
        else:                              # 重叠
            cur[1] = max(cur[1], nxt[1])   # 合并,右端点取大的

    res.append(cur)                        # 别忘了最后一个
    return res
```

### 4. 走一遍例子

```
排序后:[[1,3],[2,6],[8,10],[15,18],[16,19]]
                 ↓
cur = [1,3]

① nxt=[2,6]:2 ≤ 3 重叠,cur[1] = max(3,6) = 6 → cur=[1,6]
② nxt=[8,10]:8 > 6 不重叠,res+=[[1,6]],cur=[8,10]
③ nxt=[15,18]:15 > 10 不重叠,res+=[[8,10]],cur=[15,18]
④ nxt=[16,19]:16 ≤ 18 重叠,cur[1] = max(18,19) = 19 → cur=[15,19]

收尾:res+=[[15,19]]

答案:[[1,6],[8,10],[15,19]]  ✓
```

## 三、核心套路 2:判断区间是否重叠

两个区间 `[a, b]` 和 `[c, d]`(假设 `a ≤ c`):

| 关系 | 条件 | 例子 |
|---|---|---|
| 重叠 | `c ≤ b` | [1,5] [3,7] |
| 不重叠 | `c > b` | [1,3] [5,7] |
| 包含 | `d ≤ b` | [1,7] [2,5] |
| 相邻(接触) | `c == b+1` 或 `c == b` | [1,3] [3,5] |

**关键公式**:
- **重叠**:`a ≤ d AND c ≤ b`
- **不重叠**:`b < c OR d < a`

```python
def overlap(a, b, c, d):
    return not (b < c or d < a)   # a-d, c-b 形式
    # 等价于:a <= d and c <= b
```

## 四、核心套路 3:区间交集(LC986)

```python
def intervalIntersection(A, B):
    i, j = 0, 0
    res = []
    while i < len(A) and j < len(B):
        # 当前两个区间的交集
        lo = max(A[i][0], B[j][0])
        hi = min(A[i][1], B[j][1])
        if lo <= hi:           # 有交集
            res.append([lo, hi])

        # 谁先结束,谁往前走
        if A[i][1] < B[j][1]:
            i += 1
        else:
            j += 1
    return res
```

**双指针核心**:
- 每次算两个指针指向区间的交集
- 结束早的区间先走

## 五、核心套路 4:无重叠区间(LC435)

> 求最少移除几个区间,让剩下的都不重叠

**经典贪心**:按**右端点**排序,能选就选

```python
def eraseOverlapIntervals(intervals):
    if not intervals:
        return 0
    intervals.sort(key=lambda x: x[1])     # 按右端点排序(关键!)
    count = 1
    end = intervals[0][1]
    for i in range(1, len(intervals)):
        if intervals[i][0] >= end:         # 不重叠
            count += 1
            end = intervals[i][1]
    return len(intervals) - count          # 移除 = 总数 - 选中的
```

**为什么按右端点?**
- 选右端点小的区间,后面能放的区间更多
- 经典「活动选择问题」

## 六、核心套路 5:会议室(LC252, LC253)

### LC252:能否参加所有会议

```python
def canAttendMeetings(intervals):
    intervals.sort(key=lambda x: x[0])
    for i in range(1, len(intervals)):
        if intervals[i][0] < intervals[i-1][1]:   # 有重叠
            return False
    return True
```

### LC253:最少需要几个会议室

```python
def minMeetingRooms(intervals):
    if not intervals:
        return 0
    # 把开始时间和结束时间分开,各排个序
    starts = sorted([i[0] for i in intervals])
    ends = sorted([i[1] for i in intervals])

    rooms = 0
    end_ptr = 0
    for start in starts:
        if start < ends[end_ptr]:    # 没空房间
            rooms += 1
        else:                       # 有空房间,这个会议结束
            end_ptr += 1
    return rooms
```

**替代写法**:用最小堆

```python
import heapq
def minMeetingRooms(intervals):
    intervals.sort(key=lambda x: x[0])
    heap = []  # 存每个会议室当前会议的结束时间
    for start, end in intervals:
        if heap and heap[0] <= start:    # 有空房间
            heapq.heapreplace(heap, end)
        else:                            # 开新房间
            heapq.heappush(heap, end)
    return len(heap)
```

## 九、必须背熟的小技巧

### 1. 排序:按左端点还是右端点?

| 题型 | 排序方式 |
|---|---|
| 合并区间 | 左端点 |
| 无重叠区间(选最多) | **右端点** |
| 会议室 | 左端点 |
| 删除覆盖区间 | 左端点 |

### 2. 区间接触算不算「重叠」?

看题,常见两种:
- **算重叠**:`[1, 3]` 和 `[3, 5]` 接触 → 重叠(不冲突)
- **不算重叠**:`[1, 3]` 和 `[3, 5]` 接触 → 不算冲突(下个会议正好开始)

**判断方法**:
- 不重叠条件写 `start > end` → 接触算重叠
- 不重叠条件写 `start >= end` → 接触不算重叠

### 3. 原地修改 vs 新建结果

```python
# 合并时推荐直接修改 cur
cur[1] = max(cur[1], nxt[1])

# 或者新建元组
cur = [cur[0], max(cur[1], nxt[1])]
```

### 4. 别忘了「收尾」

合并循环结束后,**最后一个 cur 还在手上,得加到结果里**!

```python
for nxt in intervals[1:]:
    ...
res.append(cur)      # ← 千万别漏
```

### 5. 边界判断:`>` 还是 `>=`?

```python
if nxt[0] > cur[1]:       # 不重叠 → 严格大于
    ...
else:                     # 否则合并(包括相等)
    ...
```

## 十、常见陷阱

1. **漏收尾**:循环结束后最后一个区间没加进结果
2. **排序错**:用「活动选择」时按左端点排序(应该是右端点)
3. **接触边界**:相邻区间算不算合并?看题!
4. **空输入**:`intervals = []` 要特判
5. **单元素**:`len(intervals) == 1` 的情况要能跑通

## 十一、合并区间的通用模板

```python
def merge_intervals(intervals):
    if not intervals:
        return []

    # 1. 按左端点排序
    intervals.sort(key=lambda x: x[0])

    # 2. 维护当前合并区间
    res = []
    cur_start, cur_end = intervals[0]

    for start, end in intervals[1:]:
        if start > cur_end:                # 不重叠
            res.append([cur_start, cur_end])
            cur_start, cur_end = start, end
        else:                              # 重叠
            cur_end = max(cur_end, end)

    res.append([cur_start, cur_end])       # 收尾
    return res
```

**这个模板能解决 80% 的区间合并问题**,背下来!
