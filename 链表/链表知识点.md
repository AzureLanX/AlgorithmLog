## 一、链表是什么?

链表 = **节点 + 指针**,每个节点存数据和指向下一个节点的指针。

```
数组(连续):       链表(散落 + 指针):
[1][2][3][4]       [1] → [2] → [3] → [4] → None
 ↑      ↑           内存里散落各处,用指针串起来
连续内存
```

## 二、Python 中怎么表示?

### 1. LeetCode 的标准定义(单链表)

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next
```

### 2. 创建一个链表

```python
def create_list(arr):
    dummy = ListNode(0)        # 哑节点,简化操作
    cur = dummy
    for x in arr:
        cur.next = ListNode(x)
        cur = cur.next
    return dummy.next
```

### 3. 链表 vs 数组对比

| 操作 | 数组 | 链表 |
|---|---|---|
| 随机访问 `a[i]` | O(1) | O(n) |
| 头部插入 | O(n) | O(1) |
| 尾部插入 | O(1) 分摊 | O(n) |
| 中间插入 | O(n) | O(1)(已知位置) |
| 删除 | O(n) | O(1)(已知位置) |
| 内存 | 连续 | 散落 |

**核心**:链表擅长**插入/删除**,数组擅长**随机访问**。

## 三、链表的几大类型

```
单链表:  A → B → C → None
双链表:  None ← A ⇄ B ⇄ C → None
循环链表: A → B → C → A(尾指向头)
```

## 四、必须背熟的 6 大技巧

### 1. 哑节点(dummy node)

```python
dummy = ListNode(0)
dummy.next = head
# 操作 dummy.next,返回时返回 dummy.next
```

**作用**:统一处理「头节点可能被删/换」的情况,避免特判。

### 2. 快慢指针

```python
slow = fast = head
while fast and fast.next:
    slow = slow.next
    fast = fast.next.next
```

**应用**:找中点、判断环、找倒数第 k 个。

### 3. 双指针(前后指针)

```python
# 找倒数第 k 个
slow, fast = head, head
for _ in range(k):
    fast = fast.next
while fast:
    slow = slow.next
    fast = fast.next
return slow    # 倒数第 k 个
```

### 4. 头插法(反转链表)

```python
prev = None
cur = head
while cur:
    nxt = cur.next
    cur.next = prev
    prev = cur
    cur = nxt
return prev    # 新头
```

### 5. 递归(从后往前处理)

```python
def reverse(head):
    if not head or not head.next:
        return head
    new_head = reverse(head.next)
    head.next.next = head    # 后一个指向自己
    head.next = None
    return new_head
```

### 6. 哈希表(快慢指针的替代)

```python
seen = set()
cur = head
while cur:
    if cur in seen:
        return True    # 有环
    seen.add(cur)
    cur = cur.next
```

**优势**:O(n) 空间但更好懂。
**快慢指针**:O(1) 空间更优。

## 五、核心套路详解

### 套路 1:反转链表(LC206)

**题目**:反转单链表

```
1 → 2 → 3 → 4 → None
变
4 → 3 → 2 → 1 → None
```

#### 迭代版(头插法)— 推荐

```python
def reverseList(head):
    prev = None
    cur = head
    while cur:
        nxt = cur.next     # 先存下一个
        cur.next = prev    # 反转指针
        prev = cur         # prev 前进
        cur = nxt          # cur 前进
    return prev             # prev 是新头
```

**走一遍** `[1, 2, 3]`:

| 步 | cur | nxt | cur.next | prev | 状态 |
|---|---|---|---|---|---|
| 1 | 1 | 2 | None | 1 | 1 → None |
| 2 | 2 | 3 | 1 | 2 | 2 → 1 → None |
| 3 | 3 | None | 2 | 3 | 3 → 2 → 1 → None |

返回 prev=3 ✓

#### 递归版

```python
def reverseList(head):
    if not head or not head.next:
        return head
    new_head = reverseList(head.next)
    head.next.next = head
    head.next = None
    return new_head
```

**原理**:「我先反转后面的,然后把当前的拼到末尾」。

### 套路 2:合并两个有序链表(LC21)

```python
def mergeTwoLists(l1, l2):
    dummy = ListNode(0)
    cur = dummy
    while l1 and l2:
        if l1.val < l2.val:
            cur.next = l1
            l1 = l1.next
        else:
            cur.next = l2
            l2 = l2.next
        cur = cur.next
    cur.next = l1 or l2
    return dummy.next
```

**核心**:
- 哑节点简化头节点处理
- 「谁小就接谁」
- 最后把剩下的接上(无需循环)

### 套路 3:找中点(LC876)

```python
def middleNode(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    return slow
```

**为什么是 slow**?fast 走完时,fast = 2*slow 的距离。

### 套路 4:判断环(LC141)

```python
def hasCycle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow is fast:
            return True
    return False
```

**为什么相遇就有环**?fast 每次追 1 步,有环必然追上。

### 套路 5:环入口(LC142)

```python
def detectCycle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow is fast:                    # 第一步:相遇
            break
    else:
        return None
    
    slow = head                              # 第二步:slow 回到头
    while slow is not fast:
        slow = slow.next
        fast = fast.next
    return slow                              # 入口
```

**数学原理**:相遇点到入口的距离 = 头到入口的距离。

### 套路 6:倒数第 k 个(LC19)

```python
def removeNthFromEnd(head, n):
    dummy = ListNode(0, head)
    slow = fast = dummy
    for _ in range(n):
        fast = fast.next
    while fast.next:
        slow = slow.next
        fast = fast.next
    slow.next = slow.next.next
    return dummy.next
```

**核心**:fast 先走 k 步,然后 slow 跟上,fast 走完时 slow 在「倒数第 k+1」。

### 套路 7:相交链表(LC160)

```python
def getIntersectionNode(headA, headB):
    a, b = headA, headB
    while a is not b:
        a = a.next if a else headB
        b = b.next if b else headA
    return a    # None 表示不相交
```

**原理**:走完 A 走 B,走完 B 走 A,**走的总长度相同**,相交点必相遇。

### 套路 8:回文链表(LC234)

```python
def isPalindrome(head):
    # 1. 找中点
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    
    # 2. 反转后半
    prev = None
    while slow:
        nxt = slow.next
        slow.next = prev
        prev = slow
        slow = nxt
    
    # 3. 比较
    left, right = head, prev
    while right:    # 只需要走一半
        if left.val != right.val:
            return False
        left = left.next
        right = right.next
    return True
```

**核心**:
- 找中点(快慢指针)
- 反转后半
- 对比前后

## 七、必须背熟的小技巧

### 1. 哑节点:80% 的链表题都用

```python
dummy = ListNode(0, head)
# 任意操作 dummy.next
return dummy.next
```

**为什么**:「可能修改头节点」的题,不用哑节点要写一堆特判。

### 2. 删节点要拿到前一个

```python
# 错:cur 是要删的
cur = cur.next    # 漏掉一个节点,内存泄漏

# 对:prev 是要删的
prev.next = cur.next
```

### 3. 反转时先存 next

```python
nxt = cur.next     # ← 必先存!
cur.next = prev    # 这一步会丢掉 next
```

### 4. 判空要写全

```python
while fast and fast.next:    # 都要判!
    ...
```

### 5. 比较节点用 `is` 还是 `==`?

```python
# 比较值:==
if a.val == b.val:

# 比较身份(同一节点):is
if slow is fast:    # 判断环
```

## 八、链表 vs 数组:什么时候用哪个?

| 场景 | 选谁 |
|---|---|
| 频繁随机访问 | 数组 |
| 频繁头尾插入/删除 | 链表 |
| 内存敏感 | 数组(连续,无额外指针开销) |
| 大小未知/变化大 | 链表 |
| 题目明确说「链表」 | 链表 |

## 九、Python 链表小坑

### 1. `head` 可能为 `None`

```python
def func(head):
    if not head:
        return head    # ← 别忘
    ...
```

### 2. 别漏掉 `head = head.next`

```python
cur = head
while cur:
    print(cur.val)
    cur = cur.next     # ← 不然死循环!
```

### 3. 递归深度

Python 默认递归深度 1000,**超长链表递归会爆栈**。面试时优先用迭代。

### 4. 打印调试

```python
def print_list(head):
    vals = []
    while head:
        vals.append(head.val)
        head = head.next
    print(' → '.join(map(str, vals)) + ' → None')
```
